A simple biography page of Arvo Part for Web Programming class. It was an exercise to use our HTML and CSS skills. A topic was given and we were to use our skills and aesthetic design abilities to create a page that would fit the topic we chose.
P5 Project →

A midterm project for Web Programming class mainly focused on P5. A simple sketchpad is created using P5. Users are able to draw on the sketchpad. Similar to a drawing program, users are able to choose a color, enter a size stroke, clear or save the drawing they created. At the side are some example pictures for users if they want to follow along.

Web Production II Final Project

A food based website for my Web Production II class Final Project which was heavily focused on PHP and MYSQL. Main recipes change depending on the day of the week using PHP. Viewers are able to browse around the website looking for recipes and they can give ratings/feedback and/or if they would like to contact and the information they enter will be stored using database. Unfortunately, the live website is currently not working properly because I don't know how to host a PHP project live and the server I used for the database only lasted for a semester.
